This is a simple web app that takes an English sentence and converts it into a sign-language-friendly version by removing common words like "is", "the", "and", etc.
It’s built using **Python**, **Streamlit**, and **NLTK**.

step 1: Set up a virtual environment, use code:python -m venv venv
        Activate it, use code: venv\Scripts\activate
step 2: install Libraries, use code: pip install -r requirements.txt 
step 3 run the sign.py

What it does
Input:
I will be going to the market tomorrow.

Output:
going market tomorrow
